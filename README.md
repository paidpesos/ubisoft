## About:
Ubisoft is a French video game publisher, this python script uses their login api to then capture data from a account.

## Proxy Format:
`protocol://user:pass@ip:port`, `user:pass@ip:port`, and `ip:port`

## Captures:
- ubisoft username, account level, account xp, account units (currency), and games that have been ran on the account.

## Installing:

install [python](https://www.python.org/downloads/) and [git](https://git-scm.com/download/win)
```
git clone https://github.com/paidpesos/ubisoft
cd ubisoft
pip install -r requirements.txt or run `run.bat`
python main.py
```
